타닥싱크 사용법 v1.12.0
================================

캡컷 PC 프로젝트의 음성을 바탕으로 자막을 자동으로 생성하고, 내용을 수정한 뒤
캡컷 프로젝트에 바로 삽입할 수 있는 프로그램입니다.

▶ v1.12.0 업데이트: 로그인 팝업 새 디자인(연동 코드·상태를 한 창에서 표시) /
  「내 정보」(이름·이메일·코인·사용 내역) / 「코인 충전」 화면 추가(결제는 준비 중)

▶ v1.11.0 업데이트: 자막을 실제 발화 시작·끝에 스냅(에너지 온셋/오프셋) / Whisper 조기 표시 보정 강화

▶ v1.10.0 업데이트: 자동 캡션 방식 삽입 옵션 / 자막 타이밍 정밀 보정(음성 경계 자동 스냅) / 모델 다운로드 진행률 표시

※ 캡컷 초신속 수강생만 이용할 수 있습니다.
※ 강의 시작 2시간 전부터 다운로드·이용이 가능합니다.
※ 구글 로그인·기기 연동 없이는 프로그램을 사용할 수 없습니다. 미수강 계정은 로그인·연동이 거부됩니다.
※ 자세한 설명은 같은 폴더의 「사용법.txt」 또는 「사용법.md」를 참고하세요.


■ 다운로드 및 기기 연동 (타닥클래스)
-------------------------------------

【1단계: 프로그램 다운로드】
1. https://vcml.kr/subtitle-tool.html 에 접속해 구글 로그인
2. 「프로그램 다운로드」로 TadakSync.zip 받기 (강의 시작 2시간 전부터 가능)
3. zip을 원하는 폴더에 압축 해제 (zip 안에서 직접 실행하지 마세요)

【2단계: 기기 연동】
1. TadakSync.exe 실행 → 「구글 로그인」 클릭
2. 프로그램에 표시되는 연동 코드 복사
3. https://vcml.kr/subtitle-tool.html 페이지의 「앱 기기 연동」 섹션에 코드 입력
4. 「이 기기로 연동」 클릭 후 승인
5. 연동 완료 후 프로젝트 선택·자막 생성·삽입을 사용할 수 있습니다

■ 코인 안내
----------
· 최초 이용 시 100코인 지급
· 수강 후기 작성 시 100코인 추가 지급 (1회)
· 자막 생성 시 타임라인 1분당 1코인 차감 (최소 1코인)
· 로그인 후 우측 상단 「내 정보」에서 이름·이메일·보유 코인·사용 내역을 확인할 수 있습니다
· 「코인 충전」에서 충전 옵션을 미리 볼 수 있으며, 카드 결제(KG이니시스 연동)는 준비 중입니다


■ 주요 기능
----------
· 캡컷 로컬 프로젝트 자동 검색
· 프로젝트 내 영상·음성 기반 자막 자동 생성
· 한국어·일본어 등 여러 언어의 자막 생성
· 문서처럼 직접 입력하는 간편한 자막 편집
· 자막 구간별 미리듣기 / 전체 음성 연속 재생
· 재생 중인 자막 자동 표시 및 스크롤
· 자막 분할·합치기·삭제·추가
· 자막 시작 및 종료 시간 직접 수정
· 캡컷 프로젝트에 자막 트랙 바로 삽입
· 프로젝트 파일 자동 백업
· SRT 자막 파일 불러오기·보내기


■ 설치 방법
----------
1. 프로그램 압축 해제
   제공받은 TadakSync.zip을 원하는 폴더에 압축 해제합니다.
   반드시 압축을 모두 해제한 뒤 사용해 주세요.

2. 프로그램 실행
   압축을 해제한 폴더에서 TadakSync.exe 를 실행합니다.
   처음 실행 시 Windows 보안 경고가 나타날 수 있습니다.
   → 「추가 정보」 클릭 → 「실행」

3. 최초 사용 준비 (음성인식 모델 다운로드)
   처음 자막을 생성할 때 음성인식 모델(약 3.1GB)을 1회 자동 다운로드합니다.
   하단 상태 표시줄에 다운로드 진행률(%)이 표시되며, 인터넷 속도에 따라
   수 분 정도 걸릴 수 있습니다. 다운로드는 최초 1회만 진행됩니다.

   ※ 다운로드가 계속 실패하는 경우 (사내망·방화벽 등):
   1) 홈페이지에서 「음성인식 모델」 zip을 별도로 다운로드
   2) TadakSync.exe가 있는 프로그램 폴더에 바로 압축 해제 (models 폴더가 생깁니다)
      최종 경로: 프로그램 폴더\models\faster-whisper-large-v3\model.bin
   3) 프로그램을 다시 실행하면 동봉 모델을 자동 인식합니다


■ 사용 전 확인사항
----------------
· 자막을 삽입하려는 캡컷 프로젝트는 반드시 닫아 주세요.
· 캡컷 프로그램은 실행되어 있어도 되지만, 작업 대상 프로젝트가 열려 있으면
  자막 삽입이 정상적으로 완료되지 않을 수 있습니다.


■ 자막 생성 방법
--------------
1. 프로젝트 선택
   프로그램 실행 후 캡컷 프로젝트 목록에서 작업할 프로젝트를 선택합니다.

2. 자막 설정
   · 음성 인식 품질: 빠른 처리 / 균형 처리 / 정밀 처리
     (한국어 영상은 균형 이상 권장)
   · 언어: 자동 감지, 한국어, 일본어
   · 자막 한 줄 길이: 최대 글자 수 또는 어절 수 (숏폼은 짧게 권장)
   · 자동 캡션 방식: 체크 시 캡컷 「자동 캡션」과 같은 캡션 트랙으로 삽입 (기본 켜짐).
     해제하면 일반 텍스트로 삽입됩니다

3. 자막 생성
   ① 자막 생성 버튼 클릭 → 완료 후 자막이 한 줄씩 표시됩니다.


■ 자막 편집 방법
--------------
· 내용 수정: 자막 문장 클릭 후 바로 타이핑
· 분할: 나눌 위치에 커서를 두고 Enter
· 합치기: 문장 맨 앞에서 Backspace (또는 ⌃ 합치기 버튼)
· 구간 미리듣기: 각 줄 왼쪽 ▶ 버튼
· 원하는 지점부터 재생: 시간 표시 영역 클릭
· 전체 재생: 상단 ▶ 전체 재생 (재생 중 자막 자동 강조·스크롤)
· 삭제: ✕ 버튼
· 추가: + 새 자막 버튼
· 시간 수정: 시작·종료 시간 직접 입력 (시작 < 종료, 겹침 주의)


■ 캡컷 프로젝트에 자막 삽입
-------------------------
자막 확인 후 ② 캡컷 프로젝트에 삽입 클릭 → 캡컷에서 프로젝트 다시 열기
· 타임라인에 「AI 자막」 텍스트 트랙이 추가됩니다.
· 다시 삽입하면 기존 AI 자막 트랙만 새 내용으로 교체됩니다.
· 직접 만든 다른 텍스트 트랙은 유지됩니다.


■ 자막 스타일 설정
----------------
· 글자 크기: 삽입 후 캡컷에서도 변경 가능
· 세로 위치: 음수=아래, 0=중앙, 양수=위 (일반적으로 아래 권장)


■ SRT 자막
---------
· SRT보내기: 수정한 자막을 파일로 저장
· SRT 불러오기: 기존 SRT를 불러와 편집·삽입


■ 프로젝트 보호 및 백업
--------------------
삽입 전 프로젝트 파일이 자동 백업됩니다.
· 파일명: draft_content.aisub_backup_날짜.json (최근 10개 유지)
· 삽입 실패 시 자동 복구

직접 복구:
1. 캡컷에서 프로젝트 닫기
2. 프로젝트 폴더에서 백업 파일 찾기
3. 현재 draft_content.json 을 다른 곳에 보관
4. 백업 파일을 draft_content.json 으로 이름 변경 (확장자 중복 주의)
5. 캡컷에서 프로젝트 다시 열기


■ 사용 시 주의사항
----------------
· Windows용 캡컷 PC 로컬 프로젝트만 지원 (클라우드 전용 프로젝트 미지원)
· 원본 영상·음성 파일이 이동·삭제되면 해당 구간 생성 불가
· 삽입 전 작업 대상 프로젝트 반드시 닫기
· 삽입 중 프로젝트 파일 이동·수정 금지
· 프로그램 폴더 내부 파일 임의 삭제·이름 변경 금지
· 캡컷 업데이트 후 일부 기능이 동작하지 않을 수 있음
· 중요 프로젝트는 폴더 전체를 별도 백업 권장


■ 자주 묻는 질문
--------------
Q. 인터넷이 필요한가요?
A. 구글 로그인·기기 연동·코인 차감에는 인터넷 연결이 필요합니다.
   음성인식 모델은 최초 1회 다운로드 후에는 오프라인에서도 인식이 가능합니다.

Q. 모델 다운로드가 안 돼요.
A. 인터넷 연결을 확인하고 다시 시도해 주세요. 계속 실패하면 홈페이지에서
   「음성인식 모델」 zip을 받아 프로그램 폴더의 models\faster-whisper-large-v3
   폴더에 압축 해제하면 됩니다. (「설치 방법 3. 최초 사용 준비」 참고)

Q. 자막 타이밍이 영상과 미세하게 안 맞아요.
A. v1.9.0부터 실제 음성 경계에 자막을 자동으로 맞추는 정밀 보정이 적용됩니다.
   그래도 어긋난다면, 캡컷에서 음소거된 트랙 기준으로 자막을 비교하고 있는지
   확인해 보세요. 자막은 소리가 나는 트랙(음성 녹음 등)의 위치를 기준으로
   생성됩니다.

Q. 캡컷을 완전히 종료해야 하나요?
A. 프로그램은 켜져 있어도 됩니다. 작업 대상 프로젝트만 닫으세요.

Q. 기존 자막이 삭제되나요?
A. AI 자막 트랙만 교체됩니다. 직접 만든 다른 트랙은 유지됩니다.

Q. 캡컷에서 다시 수정할 수 있나요?
A. 네. 삽입된 자막은 캡컷에서 일반 텍스트처럼 수정할 수 있습니다.


■ 기본 사용 순서
--------------
1. 캡컷에서 작업 대상 프로젝트 닫기
2. 타닥싱크 실행 (로그인·기기 연동 완료 상태)
3. 프로젝트 선택
4. 언어·자막 길이 등 설정
5. ① 자막 생성
6. 자막 확인·수정 (구간/전체 재생으로 검토)
7. ② 캡컷 프로젝트에 삽입
8. 캡컷에서 프로젝트 다시 열기
9. 삽입된 자막의 서체·디자인 최종 조정
