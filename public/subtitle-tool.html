타닥싱크 2 TadakSync 2

캡컷 프로젝트의 음성을 인식해 자막을 만들고, 스타일·강조 색을 골라 캡컷 캡션 트랙에 바로 넣는 PC 프로그램입니다. 타닥클래스 구글 로그인 후 다운로드하고, 앱에서 기기 연동 뒤 코인으로 이용할 수 있습니다.

앱 기기 연동

프로그램을 실행하면 표시되는 연동 코드를 아래 입력창에 붙여넣고 기기 연동을 승인합니다.

계정당 1대 PC만 연동됩니다. 새 기기 연동 시 기존 기기는 자동 해제됩니다.

이용 안내

■ 이용 대상
  • 타닥클래스에 구글 로그인한 회원이라면 누구나 이용할 수 있습니다.
  • 계정당 PC 1대만 기기 연동됩니다.
  • 7일 동안 사용하지 않으면 기기 연동이 자동 해제됩니다. 다시 로그인해 주세요.
■ 코인 안내
  • 최초 이용 시 기본 10코인 지급
  • 이후 매일 1회 로그인(웹·앱) 시 +1코인 추가 지급 (KST 기준)
  • 전문 인식 성공 시 타임라인 1분당 1코인 차감 (최소 1코인). 무음·미인식이면 차감 없음
  • SRT만 불러와 삽입하는 경로는 코인 차감 없음
■ 사용 방법
  • 이 페이지에서 구글 로그인 → 프로그램 다운로드 (zip 전체 압축 해제 후 TadakSync2.exe 실행)
  • 앱에서 「구글 계정으로 시작하기」→ 연동 코드 확인
  • 이 페이지 「앱 기기 연동」에 코드 입력 후 승인 (계정당 1대 PC)
  • 캡컷에서 작업할 프로젝트를 닫은 뒤, 앱에서 프로젝트 선택
  • ① 전문 인식 → ② 엔터로 줄 나누기 → ③ 블록 확인·어절 강조 색 → ④ 스타일 선택 후 캡컷에 삽입
  • SRT만 불러와 삽입할 수도 있습니다 (이 경로는 코인 차감 없음)
