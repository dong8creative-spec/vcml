이용 안내

■ 이용 대상
  • 캡컷 초신속 수강생만 이용 가능합니다.
  • 강의 시작 2시간 전부터 다운로드·이용이 가능합니다.
■ 코인 안내
  • 최초 이용 허가 시 100코인 지급
  • 수강 후기 작성 시 100코인 추가 지급 (1회)
  • 자막 생성 시 타임라인 1분당 1코인 차감 (최소 1코인)
■ 사용 방법
  • 구글 로그인 후 프로그램 다운로드
  • 프로그램에서 기기 연동 코드 발급
  • 이 페이지에서 위의 "앱 기기 연동" 섹션에 코드 입력 후 연동 승인
  • 프로그램에서 자막 생성·편집 후 캡컷 프로젝트에 삽입
