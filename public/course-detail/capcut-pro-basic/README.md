# 편집 초신속 STANDARD — 상세 페이지 이미지

이 폴더에 넣은 이미지는 강의 상세 페이지 **「강의 소개」** 탭에 자동으로 표시됩니다.

## 사용 방법

1. 이 폴더에 이미지 파일을 넣습니다. (jpg, png, webp, gif — jpg/png는 webp 변환 권장)
2. **폴더에 넣은 순서대로** 상세 페이지에 표시됩니다.
3. 브라우저에서 강의 페이지를 새로고침합니다.

현재 파일: `capcut-standard-detail-01.webp`, `capcut-standard-detail-02.webp`

## 강의 페이지

`/courses/capcut-pro-basic`

## 참고

- 관리자에서 등록한 상세 이미지가 있으면 **그 이미지 뒤에** 이 폴더 이미지가 이어서 표시됩니다.
- 서버 재시작 없이 폴더에 파일만 추가·수정하면 반영됩니다.
