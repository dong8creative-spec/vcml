사용법

캡컷 초안을 고른 뒤 음성 인식 → 줄 나누기 → (선택) 번역 → 캡컷에 자막을 삽입하는 흐름입니다.

프로그램 사용 — 5단계

캡컷에서 작업할 프로젝트를 닫은 상태에서 진행하세요. 초안이 안 보이면 앱에서 「폴더 지정」을 이용하세요.
①
프로젝트 선택

캡컷 초안 목록에서 프로젝트를 고릅니다. 카드에 영상 길이·예상 코인·수정일이 표시됩니다.

②
전문 인식

언어를 선택하고 「전문 인식 시작」을 누릅니다. 음성을 분석해 전체 대본을 만듭니다.

③
엔터로 줄 나누기

인식된 전문에서 Enter 1번 = 자막 블록 1개로 나눕니다. 정식 요금은 2026년 7월 22일 오전 11시에 안내됩니다.

인식된 전문에서 Enter 1번 = 자막 블록 1개로 나눕니다. 이 단계를 사용하면 20초당 1코인이 추가로 차감됩니다.

④
번역 · 블록 확인

영어·일본어·중국어 맥락 번역(선택) 후 시간·문구를 수정합니다. 번역 코인 차감 기준은 정식 개시 시 안내됩니다.

영어·일본어·중국어 맥락 번역(선택) 후 시간·문구를 수정합니다. 번역은 20초당 10코인(1분=30코인)이 추가로 차감됩니다.

⑤
스타일 선택 · 캡컷 삽입

자막 스타일·강조 색을 고른 뒤 삽입합니다. 캡컷에 「AI 자막」 캡션 트랙이 추가됩니다.

SRT만 불러와 삽입하기

코인 차감 없음 — 빠른 경로

1단계에서 프로젝트 선택 → 「SRT 불러오기」 → 4단계(블록 확인)로 바로 이동합니다. 2·3단계(인식·줄 나누기)는 건너뜁니다.
