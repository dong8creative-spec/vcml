시작하기

타닥싱크 2를 PC에 설치하고, 타닥클래스 계정과 기기를 연동하는 방법입니다.

다운로드 & 설치

1
구글 로그인 & 프로그램 다운로드

타닥싱크 홈에서 구글 로그인 후 「프로그램 다운로드」를 눌러 zip 파일을 받습니다.

zip을 원하는 폴더에 전체 압축 해제한 뒤, zip 안에서 바로 실행하지 마세요.

2
TadakSync2.exe 실행

압축 해제한 폴더에서 TadakSync2.exe를 실행하고 「구글 계정으로 시작하기」를 누릅니다.

기기 연동

3
연동 코드 승인

앱에 표시된 연동 코드를 타닥싱크 홈의 「앱 기기 연동」에 입력하고 「이 기기로 연동」을 누릅니다.

앱에서 「브라우저에서 연동」을 누르면 코드가 포함된 페이지로 이동합니다.

기기 정책
· 계정당 PC 1대만 연동됩니다. 새 기기 연동 시 기존 기기는 자동 해제됩니다.
· 7일 미사용 시 기기 연동이 자동 해제됩니다.

음성 인식 모델

첫 전문 인식 시 음성 모델(약 3GB)이 자동으로 다운로드됩니다. 사내망·방화벽 등으로 자동 다운로드가 안 되면, 타닥싱크 홈에서 모델 파일 별도 받기 후 TadakSync2.exe가 있는 폴더에 압축 해제하세요. (models 폴더가 생깁니다)

PC 필수 구성 요소

타닥싱크 2는 Windows 전용입니다. zip에 포함된 TadakSync2.exe만으로 대부분 실행되지만, PC에 아래가 없으면 .NET 런타임 오류 등으로 실행이 안 될 수 있습니다.
필수
.NET 6 Desktop Runtime (x64)

프로그램 UI 실행에 사용합니다. Windows 10/11에 없으면 별도 설치가 필요합니다.

다운로드 — Desktop Runtime

필수
Microsoft Edge WebView2 Runtime

프로그램 화면 표시에 사용합니다. Windows 11은 대부분 기본 포함, Windows 10은 설치가 필요할 수 있습니다.

WebView2 Runtime 다운로드

권장
Visual C++ 재배포 패키지 (2015–2022)

음성 인식 엔진 등 네이티브 라이브러리 실행 시 일부 PC에서 필요합니다.

vc_redist.x64.exe 다운로드

설치 경로 팁
· 사용자 폴더·설치 경로에 한글이 있으면 .NET 6 Desktop Runtime이 특히 필요합니다.
· 가능하면 C:\TadakSync\처럼 영문만 있는 폴더에 압축 해제하세요.

설치 전 확인

· 타닥클래스 구글 로그인 회원이면 누구나 이용할 수 있습니다.
· 캡컷 PC에서 작업할 프로젝트는 닫은 상태로 타닥싱크를 실행하세요.
· 초안이 안 보이면 앱에서 「폴더 지정」으로 캡컷 초안 경로를 직접 지정할 수 있습니다.
