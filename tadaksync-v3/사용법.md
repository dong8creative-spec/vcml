# 타닥싱크 3 사용법 v3.0.0

캡컷 PC 프로젝트의 음성을 인식해 전문을 만들고, 선택적으로 영·일·중 맥락 번역 후 캡컷 캡션 트랙에 삽입합니다.

> 구글 로그인 회원 이용 가능 · 7일 미사용 시 기기 연동 자동 해제

## 시작
1. [subtitle-tool.html](https://vcml.kr/subtitle-tool.html) → 구글 로그인 → zip 다운로드
2. 압축 해제 후 `TadakSync3.exe` 실행 → 기기 연동

## 코인
| 구분 | 비용 |
|------|------|
| 최초 / 출석 | 10코인 / 매일 +1 |
| 전문 인식 | 1분당 1코인 |
| 맥락 번역 | **추가** 1분당 20코인 (약 1,000원/분, 영·일·중) |
| SRT 삽입 | 무료 |

1코인 기준 약 50원 (100코인=5,000원 충전 예정가)

예: 쇼츠 3분 인식+번역 = 3 + 60 = **63코인** (약 3,150원)

## 흐름
1. 프로젝트 선택 → 2. 전문 인식 → 3. Enter 줄 나누기  
4. 번역(선택) · 블록 확인 → 5. 스타일 · 원어/번역/둘 다 삽입  

캡컷 트랙: `AI 자막` · `AI 자막(번역)`
