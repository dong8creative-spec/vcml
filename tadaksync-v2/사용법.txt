타닥싱크 2 사용법 v2.16.2
================================

캡컷 PC 프로젝트의 음성을 인식해 전문(全文)을 먼저 보여 주고,
줄 나누기 방식을 고른 뒤 스타일·키워드 강조를 골라 캡컷 캡션 트랙에 자동 삽입하는 프로그램입니다.

※ 타닥클래스에 구글 로그인한 회원이라면 누구나 이용할 수 있습니다.
※ 구글 로그인·기기 연동 없이는 사용할 수 없습니다.
※ 7일 동안 사용하지 않으면 기기 연동이 자동 해제됩니다. 다시 로그인해 주세요.


■ 다운로드 및 기기 연동
-----------------------

【1단계: 프로그램 다운로드】
1. https://vcml.kr/subtitle-tool.html (상단 메뉴 「타닥싱크」) 접속 → 구글 로그인
2. 「프로그램 다운로드」로 zip 받기
3. zip을 원하는 폴더에 압축 해제 (zip 안에서 직접 실행하지 마세요)
4. TadakSync2.exe 실행

【2단계: 기기 연동】
1. 「구글 계정으로 시작하기」
2. 프로그램에 표시되는 연동 코드 확인
3. subtitle-tool.html 「앱 기기 연동」에 코드 입력 → 「이 기기로 연동」
4. 연동 완료 후 프로젝트 선택·인식·삽입을 사용할 수 있습니다

※ 로그인 정보는 이 PC의 %APPDATA%\TadakSync 에 저장됩니다.


■ PC 필수 구성 요소 (추가 설치)
--------------------------------
타닥싱크 2는 Windows 전용입니다. zip 안의 TadakSync2.exe만으로 대부분 실행되지만,
아래가 PC에 없으면 「.NET 런타임 오류」 등으로 실행이 안 될 수 있습니다.

【필수에 가까운 항목】
· .NET 6 Desktop Runtime (x64)
  https://dotnet.microsoft.com/download/dotnet/6.0  → 「Desktop Runtime」
· Microsoft Edge WebView2 Runtime
  https://go.microsoft.com/fwlink/p/?LinkId=2124703
  (Windows 11은 대부분 기본 포함, Windows 10은 설치가 필요할 수 있음)

【일부 PC에서 필요】
· Visual C++ 재배포 패키지 2015–2022 (x64)
  https://aka.ms/vs/17/release/vc_redist.x64.exe

【기타】
· 캡컷 PC (자막 삽입 대상)
· 인터넷 (로그인·코인·음성 모델 약 3GB 다운로드)

※ 사용자 폴더·설치 경로에 한글이 있으면 .NET 6 Desktop Runtime이 특히 필요합니다.
  가능하면 C:\TadakSync\ 처럼 영문만 있는 폴더에 압축 해제하세요.


■ 코인 안내
----------
· 최초 이용 시 기본 10코인 지급
· 이후 매일 1회 로그인(웹·앱 접속) 시 1코인 추가 지급 (KST 기준 하루 1회)
· 전문 인식 성공 시 타임라인 30초당 1코인 차감 (최소 1코인)
· 자동 어절 나누기 1회당 1코인 (어절 1~6개, 문장 경계 유지)
· 엔터 줄 나눔 1회당 2코인
· 무음·미인식이면 코인 차감 없이 자동 취소
· 전문 확정·차감 후에는 환불되지 않습니다
· SRT 불러오기 → 삽입만 하는 경로는 코인을 쓰지 않습니다


■ 사용 흐름 (5단계)
-------------------
1. 프로젝트 선택 — 캡컷 초안 목록에서 고릅니다. 해당 프로젝트는 캡컷에서 닫아 두세요.
2. 전문 인식 — 언어를 고른 뒤 인식을 시작합니다.
   · 자동 어절 나누기(1코인): 어절 최소·최대 설정 후 바로 블록 확인
   · 엔터 줄 나누기(2코인): Step 3에서 Enter 1번 = 블록 1개
3. 엔터로 줄 나누기 — (엔터 방식 선택 시만)
4. 자막 블록 확인 — 시간·문구 수정, ▶ 미리듣기, SRT 내보내기
   · 「단어·스타일 편집」 — 별도 창에서 키워드 검색·스타일·일괄 바꾸기
     (메인 자막을 보면서 작업, 창 위치·크기 자유 조절)
5. 스타일 · 삽입 — 클래식/예능/네온, 크기·위치 선택 후 캡컷 캡션 트랙에 삽입

【SRT 경로】 1단계에서 프로젝트 선택 → SRT 불러오기 → 4단계로 이동 (2·3단계 생략, 코인 없음)


■ 단어·스타일 편집 (별도 창)
----------------------------
· 4단계 진입 시 자동으로 열리거나 버튼으로 켜고 끌 수 있습니다.
· 완전 일치 / 포함 검색 → 색상·굵기·기울기 일괄 적용
· 찾은 단어 일괄 바꾸기, 스타일 해제, 되돌리기


■ 음성인식 모델
---------------
· 첫 인식 시 모델(약 3GB)이 자동 다운로드됩니다.
· 자동 다운로드가 안 되면 홈페이지 「모델 파일 별도 받기」 후
  TadakSync2.exe가 있는 폴더에 바로 압축 해제하세요. (models 폴더 생성)


■ 실행 팁
---------
1. zip을 모두 해제한 뒤 TadakSync2.exe를 실행하세요.
2. 캡컷(또는 剪映)에서 대상 프로젝트를 닫은 상태에서 삽입하세요.
3. 초안이 안 보이면 「폴더 지정」으로 초안 폴더를 직접 추가할 수 있습니다.
4. 설치 경로는 영문만 있는 폴더를 권장합니다. (예: C:\TadakSync\)
5. zip 파일 속성에서 「차단 해제」를 체크한 뒤 압축을 풀어 주세요.
6. Windows 스마트 앱 컨트롤이 차단하면 설정에서 일시적으로 끄거나,
   설치 후 다시 켜 주세요. (개별 예외 허용은 지원되지 않습니다)


■ 문의
