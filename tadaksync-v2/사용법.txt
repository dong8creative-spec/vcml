타닥싱크 2 사용법 v2.0.0
================================

캡컷 PC 프로젝트의 음성을 인식해 전문(全文)을 먼저 보여 주고,
엔터로 줄을 나눈 뒤 스타일·어절 강조 색을 골라 캡컷 캡션 트랙에 자동 삽입하는 프로그램입니다.

※ 타닥클래스에 구글 로그인한 회원이라면 누구나 이용할 수 있습니다.
※ 구글 로그인·기기 연동 없이는 사용할 수 없습니다.
※ 7일 동안 사용하지 않으면 기기 연동이 자동 해제됩니다. 다시 로그인해 주세요.


■ 다운로드 및 기기 연동
-----------------------

【1단계: 프로그램 다운로드】
1. https://vcml.kr/subtitle-tool.html (상단 메뉴 「타닥싱크」) 접속 → 구글 로그인
2. 「프로그램 다운로드」로 zip 받기
3. zip을 원하는 폴더에 압축 해제 (zip 안에서 직접 실행하지 마세요)
4. TadakSync2.exe 실행

【2단계: 기기 연동】
1. 「구글 계정으로 시작하기」
2. 프로그램에 표시되는 연동 코드 확인
3. subtitle-tool.html 「앱 기기 연동」에 코드 입력 → 「이 기기로 연동」
4. 연동 완료 후 프로젝트 선택·인식·삽입을 사용할 수 있습니다

※ 로그인 정보는 이 PC의 %APPDATA%\TadakSync 에 저장됩니다.


■ 코인 안내
----------
· 최초 이용 시 기본 10코인 지급
· 이후 매일 1회 로그인(웹·앱 접속) 시 1코인 추가 지급 (KST 기준 하루 1회)
· 전문 인식 성공 시 타임라인 1분당 1코인 차감 (최소 1코인)
· 무음·미인식이면 코인 차감 없이 자동 취소
· 전문 확정·차감 후에는 환불되지 않습니다
· SRT 불러오기 → 삽입만 하는 경로는 코인을 쓰지 않습니다


■ 사용 흐름 (5단계)
-------------------
1. 프로젝트 선택 — 캡컷 초안 목록에서 고릅니다. 해당 프로젝트는 캡컷에서 닫아 두세요.
   (카드에 영상 길이·예상 코인·수정일 표시)
2. 전문 인식 — 언어를 고른 뒤 인식을 시작합니다. (분당 코인 차감)
3. 엔터로 줄 나누기 — Enter로 자막 블록을 나눕니다.
4. 자막 블록 확인 — 시간·문구 수정, 어절 강조 색(팔레트·스포이드), ▶ 미리듣기
5. 스타일 · 삽입 — 클래식/예능/네온/배경 박스 중 선택 후 캡컷 캡션 트랙에 삽입
   (삽입 전 프로젝트 자동 백업)

【SRT 경로】 1단계에서 프로젝트 선택 → SRT 불러오기 → 4단계로 이동 (2·3단계 생략, 코인 없음)


■ 음성인식 모델
---------------
· 첫 인식 시 모델(약 3GB)이 자동 다운로드됩니다.
· 자동 다운로드가 안 되면 홈페이지 「모델 파일 별도 받기」 후
  TadakSync2.exe가 있는 폴더에 바로 압축 해제하세요. (models 폴더 생성)


■ 실행 팁
---------
1. zip을 모두 해제한 뒤 TadakSync2.exe를 실행하세요.
2. 캡컷(또는 剪映)에서 대상 프로젝트를 닫은 상태에서 삽입하세요.
3. 초안이 안 보이면 「폴더 지정」으로 초안 폴더를 직접 추가할 수 있습니다.


■ 문의
------
타닥클래스 고객센터 / 1:1 문의를 이용해 주세요.
