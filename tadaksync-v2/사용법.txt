타닥싱크 2 사용법 v2.0.0
================================

캡컷 PC 프로젝트의 음성을 Whisper로 인식해 전문(全文)을 먼저 보여 주고,
엔터로 줄을 나눈 뒤 스타일·어절 강조 색을 골라 캡컷 캡션 트랙에 자동 삽입하는 프로그램입니다.

※ 타닥싱크가 연결된 강의 수강생만 이용할 수 있습니다.
※ 강의 시작 2시간 전부터 다운로드·이용이 가능합니다.
※ 구글 로그인·기기 연동 없이는 사용할 수 없습니다.


■ 다운로드 및 기기 연동 (타닥클래스)
-------------------------------------

【1단계: 프로그램 다운로드】
1. https://vcml.kr/subtitle-tool.html (상단 메뉴 「타닥싱크」)에 접속해 구글 로그인
2. 「프로그램 다운로드」로 zip 받기 (강의 시작 2시간 전부터 가능)
3. zip을 원하는 폴더에 압축 해제 (zip 안에서 직접 실행하지 마세요)

【2단계: 기기 연동】
1. TadakSync2.exe 실행 → 「구글 계정으로 시작하기」
2. 프로그램에 표시되는 연동 코드 확인
3. https://vcml.kr/subtitle-tool.html 의 「앱 기기 연동」에 코드 입력
4. 「이 기기로 연동」 클릭 후 승인
5. 연동 완료 후 프로젝트 선택·인식·삽입을 사용할 수 있습니다

※ 로그인 정보는 이 PC의 %APPDATA%\TadakSync 에 저장됩니다.


■ 코인 안내
----------
· 일반(초신속 등) 최초 이용 시 10코인 지급 · 조회수 편집법 등 별도 강의는 강의별 안내
· 별점 5점 수강 후기 작성 시 강의별 보너스 코인(기본 50코인, 강의당 1회)
· 네이버 스마트스토어 후기는 앱 「후기 안내」에서 「작성 완료」 신고 후,
  관리자 확인을 거쳐 150코인 지급
· 전문 인식에 성공하면 타임라인 1분당 1코인 차감 (최소 1코인)
· 무음이거나 자막이 인식되지 않으면 코인 차감 없이 자동 취소됩니다
· 전문이 한 번 확정·차감된 뒤에는 환불되지 않습니다
· SRT 불러오기 → 삽입만 하는 경로는 코인을 쓰지 않습니다


■ 사용 흐름 (5단계)
-------------------
1. 프로젝트 선택 — 캡컷 초안 목록에서 고릅니다. 해당 프로젝트는 캡컷에서 닫아 두세요.
   (카드에 영상 길이·예상 코인·수정일이 표시됩니다)
2. 전문 인식 — 언어를 고른 뒤 인식을 시작합니다. (분당 코인 차감)
3. 엔터로 줄 나누기 — 나온 전문을 문장·호흡 단위로 Enter로 나눕니다.
4. 자막 블록 확인 — 시간·문구를 다듬고, 어절을 눌러 강조 색(팔레트·스포이드)을 지정할 수 있습니다.
5. 스타일 · 삽입 — 클래식/예능/네온/배경 박스 중 하나를 고른 뒤
   캡컷 캡션 트랙에 삽입합니다. 프로젝트는 자동 백업됩니다.


■ 음성인식 모델
---------------
· 첫 인식 시 모델(약 3GB)이 자동 다운로드됩니다.
· 자동 다운로드가 안 되면 홈페이지에서 「모델 파일 별도 받기」 후
  TadakSync2.exe가 있는 폴더에 바로 압축 해제하세요. (models 폴더가 생깁니다)


■ 설치·실행 팁
--------------
1. 반드시 zip을 모두 해제한 뒤 TadakSync2.exe를 실행하세요.
2. 캡컷(또는 剪映)에서 대상 프로젝트를 닫은 상태에서 삽입하세요.
3. 초안이 안 보이면 「폴더 지정」으로 초안 폴더를 직접 추가할 수 있습니다.
4. Windows 스마트 앱 컨트롤이 exe를 막는 경우, Windows 보안에서 일시적으로 끄거나
   서명된 배포본을 이용해 주세요.
