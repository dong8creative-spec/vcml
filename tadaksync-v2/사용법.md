# 타닥싱크 2 사용법 v2.0.0

캡컷 PC 프로젝트의 음성을 인식해 전문(全文)을 먼저 보여 주고, 엔터로 줄을 나눈 뒤 스타일·어절 강조 색을 골라 캡컷 캡션 트랙에 자동 삽입하는 프로그램입니다.

> 타닥클래스에 **구글 로그인한 회원**이라면 누구나 이용할 수 있습니다.  
> 구글 로그인·기기 연동 없이는 사용할 수 없습니다.  
> **7일 동안 사용하지 않으면** 기기 연동이 자동 해제됩니다. 다시 로그인해 주세요.

## 다운로드 및 기기 연동

### 1단계: 프로그램 다운로드
1. [subtitle-tool.html](https://vcml.kr/subtitle-tool.html) (상단 메뉴 「타닥싱크」) 접속 → 구글 로그인
2. 「프로그램 다운로드」로 zip 받기
3. zip을 원하는 폴더에 압축 해제 (zip 안에서 직접 실행하지 마세요)
4. `TadakSync2.exe` 실행

### 2단계: 기기 연동
1. 「구글 계정으로 시작하기」
2. 프로그램에 표시되는 연동 코드 확인
3. [subtitle-tool.html](https://vcml.kr/subtitle-tool.html) 「앱 기기 연동」에 코드 입력 → 「이 기기로 연동」
4. 연동 완료 후 프로젝트 선택·인식·삽입을 사용할 수 있습니다

로그인 정보는 이 PC의 `%APPDATA%\TadakSync`에 저장됩니다.

## 코인 안내

| 구분 | 내용 |
|------|------|
| 최초 지급 | **10코인** (회원 1회) |
| 일일 지급 | **매일 1회** 로그인(웹·앱) 시 **+1코인** (KST 기준) |
| 전문 인식 | 타임라인 **1분당 1코인** (최소 1코인) |
| 미차감 | 무음·미인식 시 자동 취소 |
| 환불 | 전문 확정·차감 후 환불 없음 |
| SRT 경로 | 불러오기·삽입만 하면 **코인 없음** |

## 사용 흐름 (5단계)

1. **프로젝트 선택** — 캡컷 초안 목록 (카드: 길이·예상 코인·수정일). 캡컷에서 해당 프로젝트는 닫아 두세요.
2. **전문 인식** — 언어 선택 후 인식 시작
3. **엔터로 줄 나누기** — Enter 1번 = 자막 블록 1개
4. **자막 블록 확인** — 시간·문구 수정, 어절 강조 색, ▶ 미리듣기, SRT 내보내기
5. **스타일 · 삽입** — 클래식/예능/네온/배경 박스, 크기·위치 선택 후 캡컷 삽입 (자동 백업)

**SRT 경로:** 1단계 프로젝트 선택 → SRT 불러오기 → 4단계 (2·3 생략, 코인 없음)

## 음성인식 모델

- 첫 인식 시 모델(약 3GB) 자동 다운로드
- 실패 시 홈페이지 「모델 파일 별도 받기」 → `TadakSync2.exe` 폴더에 압축 해제

## 실행 팁

1. zip 전체 해제 후 `TadakSync2.exe` 실행
2. 캡컷에서 대상 프로젝트를 닫은 상태에서 삽입
3. 초안이 안 보이면 「폴더 지정」으로 초안 폴더 추가
