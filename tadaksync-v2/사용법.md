# 타닥싱크 2 사용법 v2.16.2

캡컷 PC 프로젝트의 음성을 인식해 전문(全文)을 먼저 보여 주고, 줄 나누기 방식을 고른 뒤 스타일·키워드 강조를 골라 캡컷 캡션 트랙에 자동 삽입하는 프로그램입니다.

> 타닥클래스에 **구글 로그인한 회원**이라면 누구나 이용할 수 있습니다.  
> 구글 로그인·기기 연동 없이는 사용할 수 없습니다.  
> **7일 동안 사용하지 않으면** 기기 연동이 자동 해제됩니다. 다시 로그인해 주세요.

## PC 필수 구성 요소 (추가 설치)

타닥싱크 2는 Windows 전용 프로그램입니다. zip에 포함된 `TadakSync2.exe`만으로 대부분 실행되지만, 아래가 PC에 없으면 **.NET 런타임 오류** 등으로 실행이 안 될 수 있습니다.

| 구성 요소 | 필요 여부 | 설명 |
|-----------|-----------|------|
| **.NET 6 Desktop Runtime** | **대부분 필요** | 프로그램 UI(.NET) 실행에 사용. Windows 10/11에 없으면 별도 설치 |
| **Microsoft Edge WebView2 Runtime** | **대부분 필요** | 프로그램 화면 표시. Windows 11은 대부분 기본 포함, Windows 10은 설치 필요할 수 있음 |
| **Visual C++ 재배포 패키지 (2015–2022)** | 일부 PC | 음성 인식 엔진 실행 시 필요할 수 있음 |
| **캡컷 PC** | 필수 | 자막을 삽입할 대상 프로그램 |
| **인터넷** | 필수 | 로그인·코인·음성 모델(약 3GB) 다운로드 |

### 다운로드 링크

- .NET 6 Desktop Runtime: https://dotnet.microsoft.com/download/dotnet/6.0 (「Desktop Runtime」x64)
- WebView2 Runtime: https://go.microsoft.com/fwlink/p/?LinkId=2124703
- Visual C++ 재배포 패키지: https://aka.ms/vs/17/release/vc_redist.x64.exe

> **한글 사용자 폴더·설치 경로**(`C:\Users\홍길동\...` 등)에서는 .NET 6 Desktop Runtime이 특히 필요합니다. 가능하면 `C:\TadakSync\`처럼 **영문만 있는 폴더**에 압축 해제하세요.

## 다운로드 및 기기 연동

### 1단계: 프로그램 다운로드
1. [subtitle-tool.html](https://vcml.kr/subtitle-tool.html) (상단 메뉴 「타닥싱크」) 접속 → 구글 로그인
2. 「프로그램 다운로드」로 zip 받기
3. zip을 원하는 폴더에 압축 해제 (zip 안에서 직접 실행하지 마세요)
4. `TadakSync2.exe` 실행

### 2단계: 기기 연동
1. 「구글 계정으로 시작하기」
2. 프로그램에 표시되는 연동 코드 확인
3. [subtitle-tool.html](https://vcml.kr/subtitle-tool.html) 「앱 기기 연동」에 코드 입력 → 「이 기기로 연동」
4. 연동 완료 후 프로젝트 선택·인식·삽입을 사용할 수 있습니다

로그인 정보는 이 PC의 `%APPDATA%\TadakSync`에 저장됩니다.

## 코인 안내

| 구분 | 내용 |
|------|------|
| 최초 지급 | **10코인** (회원 1회) |
| 일일 지급 | **매일 1회** 로그인(웹·앱) 시 **+1코인** (KST 기준) |
| 전문 인식 | 타임라인 **30초당 1코인** (최소 1코인) |
| 자동 어절 나누기 | **1회당 1코인** (어절 1~6개 범위, 문장 경계 유지) |
| 엔터 줄 나눔 | **1회당 2코인** (전문 직접 편집) |
| 미차감 | 무음·미인식 시 자동 취소 |
| 환불 | 전문 확정·차감 후 환불 없음 |
| SRT 경로 | 불러오기·삽입만 하면 **코인 없음** |

## 사용 흐름 (5단계)

1. **프로젝트 선택** — 캡컷 초안 목록 (카드: 길이·예상 코인·수정일). 캡컷에서 해당 프로젝트는 닫아 두세요.
2. **전문 인식** — 언어 선택 후 인식 시작 → 완료 후 **줄 나누기 방식 선택**
   - **자동 어절 나누기 (1코인)** — 어절 최소·최대(1~6) 설정 후 자동 분할, Step 3 생략
   - **엔터로 줄 나누기 (2코인)** — Step 3에서 Enter 1번 = 자막 블록 1개
3. **엔터로 줄 나누기** — (엔터 방식 선택 시만) 전문 편집
4. **자막 블록 확인** — 시간·문구 수정, ▶ 미리듣기, SRT 내보내기
   - **단어·스타일 편집** — 별도 창이 열립니다. 메인 화면 자막을 보면서 키워드 검색·스타일·일괄 바꾸기 가능
5. **스타일 · 삽입** — 클래식/예능/네온, 크기·위치(인스타그램·유튜브 등) 선택 후 캡컷 삽입 (자동 백업)

**SRT 경로:** 1단계 프로젝트 선택 → SRT 불러오기 → 4단계 (2·3 생략, 코인 없음)

### 단어·스타일 편집 (Step 4, 별도 창)

- 4단계 진입 시 자동으로 열리거나, 「단어·스타일 편집」 버튼으로 켜고 끌 수 있습니다.
- **별도 OS 창**이라 메인 자막 목록을 가리지 않습니다. 창 위치·크기는 자유롭게 조절하세요.
- 키워드 **완전 일치** / **포함** 검색 → 색상·굵기·기울기 일괄 적용
- 찾은 단어 **일괄 바꾸기**, 스타일 해제, 되돌리기 지원

## 음성인식 모델

- 첫 인식 시 모델(약 3GB) 자동 다운로드
- 실패 시 홈페이지 「모델 파일 별도 받기」 → `TadakSync2.exe` 폴더에 압축 해제

## 실행 팁

1. zip 전체 해제 후 `TadakSync2.exe` 실행
2. 캡컷에서 대상 프로젝트를 닫은 상태에서 삽입
3. 초안이 안 보이면 「폴더 지정」으로 초안 폴더 추가
4. 설치 경로는 **영문만 있는 폴더**를 권장합니다 (예: `C:\TadakSync\`)
5. zip **속성 → 차단 해제** 후 압축 해제
6. **Windows 스마트 앱 컨트롤**이 차단하면 설정에서 일시적으로 끄고 설치 후 다시 켜 주세요
